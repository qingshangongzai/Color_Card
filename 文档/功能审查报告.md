> 生成日期：2026-05-09

***

## 一、项目概览

| 项目   | 信息                                                    |
| ---- | ----------------------------------------------------- |
| 应用名称 | 取色卡 (Color Card)                                      |
| 当前版本 | v1.10.0 Beta 1                                        |
| 技术栈  | PySide6 + qfluentwidgets + PySideSix-Frameless-Window |
| 运行平台 | Windows / macOS                                       |

| 应用定位 | 一站式配色工具和图片色彩分析工具                                      |

***

## 二、启动流程（main.py）

`main.py` 作为应用入口，执行以下启动序列：

1. **AppUserModelID 设置** — 在 `QApplication` 创建前调用 Win32 API，确保 Windows 任务栏正确分组
2. **高 DPI 策略** — `PassThrough` 模式，避免 Qt 缩放截断
3. **日志系统初始化** — 双输出（文件+控制台），自动清理 30 天旧日志
4. **全局异常处理器** — 捕获未处理异常写入日志
5. **启动画面** — 加载 ICO 图标居中显示，附带"启动中……"提示
6. **stdout 重定向** — 临时屏蔽 qfluentwidgets 推广提示
7. **Qt 消息处理器** — 过滤 `QFont::setPointSize` 警告
8. **配置加载** — JSON 配置，支持损坏自动备份
9. **语言初始化** — 支持 7 种语言，TOML 翻译文件
10. **主题设置** — 浅色/深色/跟随系统，主题色 `#0078d4`
11. **主窗口创建** — 延迟导入各功能界面
12. **任务栏图标修复** — Win32 API 确保任务栏图标正确显示
13. **窗口置前** — 解决启动期间窗口被遮挡问题
14. **自动更新检查** — 延迟 3 秒，每周一次，从 Gitee API 获取最新版本

## 四、架构设计特点

| 特点       | 说明                                                                        |
| -------- | ------------------------------------------------------------------------- |
| 延迟加载     | 界面类按需创建，重量级服务通过工厂函数延迟导入，减少启动时间                                            |
| 异步服务     | 所有耗时操作（图片加载/主色调提取/直方图计算/导入导出）均使用 QThread + Signal，QueuedConnection 保证线程安全 |
| LRU 缓存   | 统一 BaseCache 基类，配色/直方图/影调分析各自独立缓存，图片内存管理器 500MB 上限                        |
| NumPy 优化 | 明度计算/直方图计算/高亮遮罩生成均使用 NumPy 向量化                                            |
| 多色彩空间    | 5 种色彩空间精确转换，ICC 配置文件支持                                                    |
| 双色轮模式    | RGB 色轮（数字设计）+ RYB 色轮（传统美术）                                                |
| 信号驱动     | 设置变更通过 Signal 广播到所有功能界面，跨面板图片同步                                           |
| 主题适配     | ~40 个统一颜色函数 + themeChangedFinished 信号自动更新                                 |
| 国际化      | 7 种语言，TOML 翻译文件，language\_changed 信号动态切换                                  |
| 无边框窗口    | BaseFramelessDialog 统一基类，淡入动画避免深色模式闪烁                                     |
