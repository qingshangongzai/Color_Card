# 明度分析功能整合方案

> 将独立的明度分析 Demo 整合进 Color Card 主项目

## 一、功能定位

在「明度分析」面板的右键菜单中添加「影调分析」选项，点击后弹出大对话框显示分析结果。

**交互流程**：
1. 用户在明度分析面板加载图片
2. 右键点击画布
3. 选择「明度分析」菜单项
4. 弹出明度分析对话框，显示当前图片的影调分析结果

## 三、实现步骤

### 步骤 1：创建核心服务

创建 `core/tone_analysis.py`：

```python
"""影调分析服务"""

from dataclasses import dataclass
import numpy as np
from PIL import Image


@dataclass
class ToneAnalysisResult:
    """影调分析结果"""
    mean: float
    median: float
    std: float
    min_val: int
    max_val: int
    shadows: float
    midtones: float
    highlights: float
    tone_type: str
    histogram: np.ndarray


class ToneAnalysisService:
    """影调分析服务"""
    
    def analyze_from_array(self, img_array: np.ndarray) -> ToneAnalysisResult:
        """从 NumPy 数组分析影调
        
        Args:
            img_array: RGB 图片数组 (H, W, 3)
            
        Returns:
            ToneAnalysisResult: 分析结果
        """
        # 转换为灰度 (标准亮度公式)
        gray = (0.299 * img_array[:,:,0] + 
                0.587 * img_array[:,:,1] + 
                0.114 * img_array[:,:,2]).astype(np.uint8)
        
        # 计算统计信息
        mean = float(np.mean(gray))
        median = float(np.median(gray))
        std = float(np.std(gray))
        min_val = int(np.min(gray))
        max_val = int(np.max(gray))
        
        # 计算分区占比
        shadows = float(np.sum(gray < 64) / gray.size * 100)
        midtones = float(np.sum((gray >= 64) & (gray < 192)) / gray.size * 100)
        highlights = float(np.sum(gray >= 192) / gray.size * 100)
        
        # 判断影调类型
        tone_type = self._classify_tone(mean, shadows, highlights, std, midtones)
        
        # 计算直方图
        hist, _ = np.histogram(gray, bins=256, range=(0, 256))
        
        return ToneAnalysisResult(
            mean=mean, median=median, std=std,
            min_val=min_val, max_val=max_val,
            shadows=shadows, midtones=midtones,
            highlights=highlights, tone_type=tone_type,
            histogram=hist
        )
    
    def _classify_tone(self, mean, shadows, highlights, std, midtones) -> str:
        """判断影调类型"""
        if mean > 180 and shadows < 5:
            return "高调 (High-key)"
        elif mean < 80 and highlights < 5:
            return "低调 (Low-key)"
        elif 110 <= mean <= 150 and 40 <= midtones <= 70:
            return "中调 (Mid-tone)"
        elif std < 40:
            return "短调 - 低对比度"
        elif std > 80:
            return "长调 - 高对比度"
        else:
            return "正常调"
```

```python
def _update_styles(self):
    if isDarkTheme():
        self.figure.set_facecolor('#1e1e1e')
        # 深色主题配色
    else:
        self.figure.set_facecolor('#ffffff')
        # 浅色主题配色
    self.canvas.draw()
```
