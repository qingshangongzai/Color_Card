### 任务 3.6：直方图计算优化

**目标**：优化直方图计算性能，提升大图片加载体验

**文件**：修改 `core/color.py`、`ui/histograms.py`

**当前问题**：

- 直方图计算在主线程执行，阻塞UI
- 大图片（>4000px）计算耗时 2-3 秒

**优化方案**：


1. **后台线程计算**：
   - 创建 `HistogramCalculator` 工作线程
   - 图片加载完成后异步计算直方图
   - 计算完成后通过信号更新UI
2. **延迟计算**：
   - 图片显示后延迟 500ms 再开始计算
   - 用户可立即看到图片，感知更流畅
3. **采样优化**（可选）：
   - 大图片使用更大的采样步长（step=8）
   - 或使用缩略图计算直方图

**实现内容**：

```python
class HistogramCalculator(QThread):
    """后台直方图计算线程"""
    finished = Signal(list)  # 计算完成信号
    
    def __init__(self, image: QImage, sample_step: int = 4):
        super().__init__()
        self._image = image
        self._sample_step = sample_step
    
    def run(self):
        histogram = calculate_histogram(self._image, self._sample_step)
        self.finished.emit(histogram)
```

**已完成内容**：

- ✅ 新建 `core/histogram_service.py` 模块，包含 `HistogramService` 和 `HistogramCalculator`
- ✅ `HistogramCalculator` 在后台线程执行直方图计算（明度、RGB、色相）

### 任务 4.2：缓存优化 ✅

**目标**：通过缓存机制减少重复计算，提升响应速度

**当前问题**：

- 相同色相的配色计算结果未缓存
- 直方图数据在切换标签页后丢失

**优化方案**：


1. **配色计算缓存**：

```python
class ColorSchemeCache:
    """配色计算结果缓存"""
    
    def __init__(self, max_size: int = 100):
        self._cache = {}  # {(scheme_type, hue, count, saturation): colors}
        self._max_size = max_size
    
    def get(self, scheme_type: str, hue: float, count: int, saturation: float) -> Optional[list]:
        key = (scheme_type, round(hue, 1), count, saturation)
        return self._cache.get(key)
    
    def set(self, scheme_type: str, hue: float, count: int, saturation: float, colors: list):
        if len(self._cache) >= self._max_size:
            # LRU 淘汰
            oldest = next(iter(self._cache))
            del self._cache[oldest]
        key = (scheme_type, round(hue, 1), count, saturation)
        self._cache[key] = colors
```


2. **直方图数据缓存**：

- 图片未变化时复用直方图数据

```python
class ImageMemoryManager:
    """图片内存管理器"""
    
    def __init__(self, max_memory_mb: int = 500):
        self._max_memory = max_memory_mb * 1024 * 1024
        self._current_memory = 0
        self._images = {}  # {source_id: (pixmap, size)}
    
    def add_image(self, source_id: str, pixmap: QPixmap):
        size = pixmap.sizeInBytes()
        # 超过限制时清理旧图片
        while self._current_memory + size > self._max_memory and self._images:
            self._evict_oldest()
        self._images[source_id] = (pixmap, size)
        self._current_memory += size
```
