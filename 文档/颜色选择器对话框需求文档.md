## 1. 需求概述

### 1.1 背景

在编辑配色对话框中，用户需要通过输入 HEX 值来指定颜色，这种方式不够直观。为了提升用户体验，需要提供一个可视化的颜色选择器，允许用户通过点击、拖动等方式快速选择颜色。

### 1.2 目标

在 `edit_palette.py` 中实现一个完整的颜色选择器对话框，用户可通过右键点击颜色预览块触发，提供类似 Windows 画图颜色选择器的简化功能。

### 1.3 触发方式

- **触发源**：`ColorInputRow` 中的 `preview_block`（颜色预览块）
- **触发动作**：右键点击直接打开颜色选择器对话框
- **交互方式**：右键点击预览块立即弹出 `ColorPickerDialog`

## 2. 功能需求

### 2.1 核心功能清单

| 功能模块 | 功能描述 | 优先级 |
|---------|---------|--------|
| 基本颜色网格 | 48个常用颜色预设，点击即可选择 | 高 |
| 颜色预览块 | 实时显示当前选择的颜色 | 高 |
| 颜色模式滑块 | 两个颜色模式各3个滑块（带颜色渐变） | 高 |
| 数值同步 | 所有输入方式实时同步 | 高 |
| HEX 输入 | 输入框输入 #RRGGBB 格式（由设置控制显示） | 高 |
| 确认/取消 | 标准对话框按钮，确认后返回颜色值 | 高 |

### 2.2 功能详细说明

#### 2.2.1 基本颜色网格

- **布局**：6行 × 8列，共48个颜色块
- **尺寸**：每个色块 24×24 像素，间距 4 像素
- **颜色定义**：参考 Qt 颜色选择器标准预设色板
- **交互**：单击选择，选中后高亮显示边框

**预设颜色值**：
```
第1行: #000000 #7F7F7F #880015 #ED1C24 #FF7F27 #FFF200 #22B14C #00A2E8
第2行: #3F48CC #A349A4 #FFFFFF #C3C3C3 #B97A57 #FFAEC9 #FFC90E #EFE4B0
第3行: #99D9EA #7092BE #C8BFE7 #C3C3C3 #B97A57 #FFAEC9 #FFC90E #EFE4B0
第4行: #99D9EA #7092BE #C8BFE7 #C3C3C3 #B97A57 #FFAEC9 #FFC90E #EFE4B0
第5行: #99D9EA #7092BE #C8BFE7 #C3C3C3 #B97A57 #FFAEC9 #FFC90E #EFE4B0
第6行: #99D9EA #7092BE #C8BFE7 #C3C3C3 #B97A57 #FFAEC9 #FFC90E #EFE4B0
```

#### 2.2.2 颜色预览块

- **尺寸**：60×60 像素
- **位置**：右半部分顶部
- **显示**：当前选择的颜色
- **更新方式**：实时更新

#### 2.2.3 颜色模式滑块

**模式选择**：
- 显示的颜色模式由设置统一管理（`settings.color_modes`）
- 设置中"色彩模式显示"选项决定显示哪两个模式
- 可选模式：HSB、LAB、HSL、CMYK、RGB

**每个模式的滑块**：
- 每个模式包含3个参数滑块
- 滑块带颜色渐变背景（根据当前颜色动态计算）
- 拖动滑块时实时更新颜色

**滑块规格**：
- 宽度：200像素
- 高度：20像素
- 滑块指示器：圆形，直径16像素
- 数值标签：滑块右侧显示当前值

**各模式参数范围**：

| 模式 | 参数1 | 参数2 | 参数3 |
|-----|-------|-------|-------|
| HSB | H: 0-360° | S: 0-100% | B: 0-100% |
| RGB | R: 0-255 | G: 0-255 | B: 0-255 |
| LAB | L: 0-100 | A: -128-127 | B: -128-127 |
| HSL | H: 0-360° | S: 0-100% | L: 0-100% |
| CMYK | C: 0-100% | M: 0-100% | Y: 0-100% |

#### 2.2.4 HEX 输入框

- **显示控制**：由设置 `settings.hex_visible` 控制是否显示
- **样式**：使用配色管理内的样式（与 `ColorInputRow` 中的 HEX 输入框一致）
- **格式**：#RRGGBB
- **自动补全**：自动补全 # 前缀
- **实时同步**：输入时实时更新其他组件

### 2.3 数值同步规则

**内部状态存储**：以 RGB 作为主状态

**同步流程**：
```
用户点击预设颜色
        ↓
更新内部 RGB 状态
        ↓
计算所有模式数值
        ↓
同步更新所有滑块位置
        ↓
更新预览块
```

```
用户拖动滑块
        ↓
根据当前模式和滑块值计算 RGB
        ↓
更新内部 RGB 状态
        ↓
同步更新其他模式滑块
        ↓
更新预览块
```

### 5.4 滑块渐变算法

**HSB 模式滑块渐变**：
```python
# H 滑块：色相渐变（彩虹色）
h_gradient = QLinearGradient(left, top, right, top)
for i in range(7):
    hue = i * 60  # 0, 60, 120, 180, 240, 300, 360
    color = QColor.fromHsv(hue, 255, 255)
    h_gradient.setColorAt(i / 6, color)

# S 滑块：从灰到当前色相的纯色
s_gradient = QLinearGradient(left, top, right, top)
s_gradient.setColorAt(0.0, QColor.fromHsv(h, 0, b))
s_gradient.setColorAt(1.0, QColor.fromHsv(h, 255, b))

# B 滑块：从黑到当前色相的纯色
b_gradient = QLinearGradient(left, top, right, top)

b_gradient.setColorAt(1.0, QColor.fromHsv(h, s, 255))
```

## 7. 集成方案

### 7.1 修改 ColorInputRow

在 `ColorInputRow` 类中添加右键事件处理：

```python
class ColorInputRow(QWidget):
    # ... 现有代码 ...
    
    def setup_ui(self):
        # ... 现有代码 ...
        
        # 颜色预览块
        self.preview_block = QLabel()
        self.preview_block.setFixedSize(28, 28)
        self._update_preview_style(None)
        # 安装事件过滤器以捕获右键点击
        self.preview_block.installEventFilter(self)
        layout.addWidget(self.preview_block)
        
        # ... 其他代码 ...
    
    def eventFilter(self, obj, event):
        """事件过滤器处理右键点击"""
        if obj == self.preview_block and event.type() == QEvent.Type.MouseButtonPress:
            if event.button() == Qt.MouseButton.RightButton:
                self._open_color_picker()
                return True
        return super().eventFilter(obj, event)
    
    def _open_color_picker(self):
        """打开颜色选择器对话框"""
        # 获取当前颜色作为初始值
        if self._color_info and 'rgb' in self._color_info:
            initial = self._color_info['rgb']
        else:
            initial = (128, 128, 128)  # 默认灰色
        
        dialog = ColorPickerDialog(initial, self)
        if dialog.exec() == QDialog.DialogCode.Accepted:
            color_info = dialog.get_color_info()
            self.set_color_info(color_info)
```
